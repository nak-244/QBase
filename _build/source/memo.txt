# ■ 手順（そのままコピペ）

## ① 既存コンテナ削除（念のため）

```bash
docker rm -f open-webui
```

---

## ② 最新版を起動

```bash
docker run -d \
  --restart unless-stopped \
  -p 4000:8080 \
  -v open-webui:/app/backend/data \
  --name open-webui \
  ghcr.io/open-webui/open-webui:main
```

---

# ■ ③ アクセス

👉 [http://localhost:4000](http://localhost:4000)

---

# ■ これでどうなるか

* 最新版OpenWebUIが起動
* 初期セットアップ画面が出る
* ユーザー作成
