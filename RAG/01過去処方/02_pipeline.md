# パイプライン
## ① ファイル監視
`Path.rglob()`で共有フォルダ巡回。
`mtime` `size` `hash` で差分検出。
→ 更新されたExcelだけ発見。

## ② Excel変換
```
Sheet1
 ├ 顧客
 ├ 商品
 └ 売上
 ```
このデータを、
```
# 顧客情報

顧客名：○○
担当：△△

# 売上

2026年：
```
に変換ができる。

## ③ AI向け構造化
単純な文字抽出ではなく、`意味構造を整理`できます。
`if "顧客" in sheet_name:`
のように分岐ができるため、 ***業務ルール込みのRAG化*** ができる。


## ④ Markdown作成
Pythonで `with open(...,"w")`するだけで自動変換できる。

## ⑤ OpenWEBUI API投入
OpenWEBUIは、内部的に`HTTP API`で動いています。
Pythonから`requests.post()`でファイルの登録ができるため、***GUIが不要***となります。

## ⑥ 差分更新
Pythonは`changed_files.json`のような状態管理ができる。
そのため、***更新ファイルだけ再登録***が可能。

### ⑦ 古いRAG削除
```元ファイル削除→RAG側も削除```まで自動化が可能。
***同期型RAG***となります。

### ⑧ スケジュール実行
Windowsなら、タスクスケジューラーで制御可能。
```1時間ごと``` ```5分ごと``` ```毎日深夜``` などで自動実行ができるので、***放置運用***できます。
