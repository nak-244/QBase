# Ollama 並列処理
## ① Ollamaで環境変数対応
Ollamaの最新リリースでは、次の2つの環境変数を設定することで、新しい並行機能を利用できます。
1. OLLAMA_NUM_PARALLEL: 単一のモデルに対して複数のリクエストを同時に処理する機能
2. OLLAMA_MAX_LOADED_MODELS: 複数のモデルを同時にロードする機能

これらの機能を有効にするためには、以下の環境変数を設定して ollama serve を実行します。
```
OLLAMA_NUM_PARALLEL=4 OLLAMA_MAX_LOADED_MODELS=4 ollama serve
```

## ② vLLM？
- 同時接続ユーザー数 < 5人 → Ollamaで十分
- 同時接続ユーザー数 5〜50人 → vLLMを検討
- GPUメモリの使用率を最適化したい → vLLM推奨
- GPUがない / MacBook Airで動かしたい → Ollama一択
