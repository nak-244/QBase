cloudflared tunnel create qbase-tunnel

    json保存先　/home/qpc/.cloudflared/9ceac628-dcb0-44b0-ad63-016abade9feb.json
    id 9ceac628-dcb0-44b0-ad63-016abade9feb


nano ~/.cloudflared/config.yml
下記を追加
    tunnel: qbase-tunnel
    credentials-file: /home/qpc/.cloudflared/9ceac628-dcb0-44b0-ad63-016abade9feb.json

    ingress:
    - hostname: qbase.qbiworld.com
        service: http://localhost:4000
    - service: http_status:404

ローカルで起動して動作確認
cloudflared tunnel run qbase-tunnel
    tunnelID=9ceac628-dcb0-44b0-ad63-016abade9feb
    このIDをCNAME登録
